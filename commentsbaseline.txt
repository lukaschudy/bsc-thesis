Baseline & Grid Robustness — Key Results and Interpretation
=============================================================

BASELINE REPLICATION (m=15)
-----------------------------
Parameters: alpha=0.15, beta_eff=4e-6 (MExpl=0.02, itersPerEpisode=5000),
            delta=0.95, logit demand, n=2 agents, k=1 memory, 1000 sessions.

Result: avgPrGain (Delta) = 0.848
  - Agent 1: 0.844
  - Agent 2: 0.852
  - All 1000 sessions converged.

This matches Calvano et al. (2020) Table I representative experiment (~0.84).
The C++ implementation is validated.


GRID ROBUSTNESS CHECK (m=25)
------------------------------
Same parameters, only Number of Prices increased from 15 to 25.
Beta held fixed (same MExpl=0.02) — this is Calvano's original approach,
not the nu-corrected version.

Result: avgPrGain (Delta) = 0.806
  - Agent 1: 0.806
  - Agent 2: 0.807
  - All 1000 sessions converged.


GRID ROBUSTNESS CHECK (m=50)
------------------------------
Same parameters, only Number of Prices increased from 15 to 50.
Beta held fixed. 250 sessions (down from 1000 — still enough for a
stable Delta estimate; see design_choices.txt).

Result: avgPrGain (Delta) = 0.707  (SE = 0.023)
  - All 250 sessions converged.
  - Average convergence time: 476 episodes.


GRID ROBUSTNESS CHECK (m=100)
-------------------------------
Same parameters, Number of Prices = 100.
Max episodes bumped 1000 -> 5000 to give m=100 a fair chance to converge
(state space is 100^2 = 10,000 cells, so convergence needs more episodes).
250 sessions.

Result: avgPrGain (Delta) = 0.706  (SE = 0.016)
  - All 250 sessions converged.
  - Average convergence time: 1144 episodes (well within the 5000 cap).

INTERPRETATION OF THE GRID-ROBUSTNESS RESULTS
----------------------------------------------
(1) Collusion survives grid refinement.
    Delta stays well above 0.70 even at m=100. Refining the price grid
    by 6.67x (15 -> 100) shaves off ~17% of Delta but does not break
    the result. The Calvano phenomenon is not a grid-resolution artifact.

(2) Delta flattens between m=50 and m=100 (0.707 -> 0.706).
    The curve has nearly asymptoted. Going to m=200 or m=500 would
    likely change very little. The "true" continuous-price Delta is
    probably around ~0.70 — still firmly in the collusion regime.

(3) m=100 converged on a local machine without the cluster.
    Average time-to-convergence was 1144 episodes, comfortably below
    the bumped 5000-episode cap. The RUG cluster remains useful for
    larger sensitivity sweeps but is NOT required for this baseline
    result.

(4) Sampling noise shrinks at finer grids (SE 0.11 -> 0.02).
    Opposite of naive expectation. Apparently at finer grids the
    converged profit gain is more consistent across sessions —
    probably because there are more nearby prices agents can
    coordinate on, so different sessions land on similar Deltas.

(5) Caveat re: Calvano's original confound (see grid_robustness_note.txt).
    These runs use fixed beta (Calvano's original approach). The nu-
    corrected beta version — which holds learning quality constant
    across m — has NOT been run yet; flagged for future work.


WHAT DELTA MEANS
-----------------
Delta = (pi_bar - pi_Nash) / (pi_Monopoly - pi_Nash)

  Delta = 0  ->  firms earn exactly Nash (competitive) profit
  Delta = 1  ->  firms earn exactly monopoly profit

So Delta = 0.848 means firms end up 84.8% of the way from Nash to monopoly.
Delta = 0.806 means 80.6% of the way. In plain terms: prices are still far
above the competitive level, just very slightly less so with more price options.


INTERPRETATION
---------------
The drop from 0.848 to 0.806 going from m=15 to m=25 is modest (~5 pp).
Collusion persists and remains substantial with a finer price grid.

Intuition: with more price options, agents find it slightly harder to
implicitly coordinate on a focal price point, but they still manage to
sustain prices well above Nash. This is consistent with Calvano's own
finding that Delta decreases with m but stays substantial even at m=100 (~70%).

The m=25 result sits solidly in Case A (from grid_robustness_note.txt):
collusion is genuine, not an artifact of the coarse 15-price grid.

Caveat: beta is held fixed, so part of the drop at m=25 could reflect
Calvano's exploration confound (less exploration per Q-cell with larger
state space) rather than a harder coordination problem per se. The
nu-correction methodology in grid_robustness_note.txt describes how to
disentangle these effects if needed.


CONCLUSION FOR THE THESIS
---------------------------
The baseline collusion finding is robust to a finer price grid (m=25).
We proceed with m=15 for the friction experiments to keep the state space
tractable, with this robustness check as supporting evidence that the
baseline result is not driven by the coarseness of the grid.
